# **App Name**: NoorRamadan

## Core Features:

- Authentication: Secure user authentication using email and password, with registration, login, and password reset functionality.
- Home Dashboard: Display prayer times (including Iftar) and a daily Dua, and provide quick access to other features like Quran reading and learning Namaz.
- City & Timings: Allow users to select their city and view prayer times for that location.  The selected city is stored in the user's Firestore profile.
- Quran Reader: Enable users to read the Quran with multi-language translations and transliteration, with features for bookmarking and favoriting verses.
- Duas: Provide a curated collection of Duas for various occasions, with translations, transliteration, and the option to favorite.
- Learn Namaz: Offer a step-by-step guide to performing Salah, with progress tracking and a Kids Mode for simplified learning.
- Personalized Recommendations: Based on user's reading habits and saved duas, recommend specific verses or duas, helping them engage more meaningfully with the app. The app uses a tool to decide when to surface relevant information.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) for a premium, modern feel.
- Background color: Very light grey (#F5F5F5).
- Accent color: Violet (#8F00FF), to bring attention to key UI elements.
- Body and headline font: 'Alegreya', a serif font with an elegant, intellectual feel. Note: currently only Google Fonts are supported.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, elegant icons that are easily understandable. Consider Islamic geometric patterns or crescent moon imagery.
- Mobile-first design with a bottom navigation bar for key features. Utilize cards with soft shadows and rounded corners for a clean, organized layout.
- Subtle animations and transitions to provide a smooth and engaging user experience. For example, smooth scrolling and fading effects.
